# DXP

Welcome to the official source of documentation for the latest versions of Liferay DXP and Liferay Portal! Liferay DXP is the commercially supported and enterprise ready platform used around the world, built on the foundation of Liferay Portal. If this is your first time learning about Liferay DXP, get started quickly with our [Starting with a Docker Image](./getting-started/starting-with-a-docker-image.md) article. Unless otherwise noted, content on this site is applicable to both Liferay DXP and Liferay Portal.
